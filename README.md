## HuaiNianPlayer

一个使用React-Native构建的Android在线音乐播放器。

## 功能

提供了常见的在线播放器的基本功能

+ 支持乱序播放、单曲循环、列表循环。
+ 支持通知栏和锁屏界面对播放器的控制
+ 支持查看历史记录
+ 支持专辑的删改
